Vigilo is a web-based hacking simulation game, where you play the role of a hacker seeking for money and power. 

You have a lot of things to do in-game:
  - Complete Missions
  - Earn money
  - Install viruses
  - Hack players
  - Mine bitcoins
  - Upgrade hardware
  - Research Software
  - Create a clan / guild
  - DDoS Attacks
